# Meus Projetos

## [Disney+ EDA - Kaggle](https://www.kaggle.com/code/lucaselpidio/disneyplus-eda)

Projeto acadêmico com objetivo de aplicar conceitos fundamentais de EDA em um Dataset público contendo os títulos disponíveis na plataforma Disney+, buscando entender a distribuição entre filmes e séries, observar padrões de lançamento ao longo dos anos, gerando visualizações simples e informativas. 

Construído utilizando linguagem Python, etapa de ETL construída com pacote Pandas. 

Visualização de Dados utilizando pacotes Seaborn / Matplotlib , todas as etapas foram construídas com foco didático, consolidando conhecimentos de análise exploratória e visualização de dados.

## [Análise e Predição de Aluguéis de Bicicletas em Londres - Kaggle](https://www.kaggle.com/code/lucaselpidio/an-lise-e-predi-o-de-alugu-is-de-bicicletas)

Projeto acadêmico explorando padrões sazonais, meteorológicos e temporais que influenciam o número de aluguéis de bicicletas, além de testar um modelo preditivo básico de Machine Learning. 

Resgatamos um dataset público no Kaggle, utilizando como base a linguagem Python e seus pacotes Pandas e Seaborn na construção da EDA. Buscamos entender possíveis relações entre colunas, auxiliando na etapa de construção do modelo de predição. 

Fixamos conhecimentos em EDA, ETL, visualização de dados com Seaborn, finalizando na construção de um modelo de ML como regressão linear.

## [Pipeline Dados Telegram - Kaggle](https://www.kaggle.com/code/lucaselpidio/pipeline-dados-telegram-projeto-final-ebac)

Neste projeto acadêmico buscamos desenvolver um pipeline de dados completo para coletar, transformar e analisar mensagens de um bot do Telegram, com foco em compreender padrões de uso e comportamento em grupos de conversas. 

Transformamos dados transacionais provenientes da API do Telegram em dados analíticos que podem ser consultados via SQL e visualizados em Dashboards. 

Consolidamos conhecimentos em vários serviços da AWS, como armazenamento otimizado no formato parquet diretamente no S3, Lambda executando funções de ETL, API Gateway redirecionando para outros serviços da AWS e por fim Event Bridge realizando automação do processo de ETL diariamente.

## [Credit EDA -  AWS S3 + AWS Athena - Kaggle](https://www.kaggle.com/code/lucaselpidio/credit-eda)

Projeto acadêmico com objetivo de compreender padrões de gasto e identificar o perfil do público-alvo para futuras campanhas de marketing, a EDA considera diferentes faixas etárias, buscando revelar tendências comportamentais e insights que possam apoiar decisões estratégicas relacionadas a oferta de produtos e serviços financeiros. 

A ideia é utilizar uma amostra de um dataset fictício para consolidar conhecimentos de ferramentas AWS e consultas SQL auxiliando na EDA. Foi realizado a construção de tabelas SQL resgatando dados do S3, com auxílio do Athena para realizar as consultas gerando insights valiosos.
